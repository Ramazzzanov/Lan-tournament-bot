
# LAN Tournament Telegram Bot

Features:
- PUBG / Dota 2 / CS2 tournaments
- Team registration
- Admin panel
- PostgreSQL support
- Aiogram 3

## Run

```bash
pip install -r requirements.txt
python main.py
```

## ENV

Create `.env`

```env
BOT_TOKEN=YOUR_TOKEN
ADMIN_ID=YOUR_TELEGRAM_ID
DATABASE_URL=sqlite:///database.db
```
